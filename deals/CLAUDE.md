# Deals Folder — Instructions

This folder contains active deal workspaces. Each deal gets its own subfolder named by company (e.g., `deals/acme-corp/`).

## Deal Folder Structure
```
deals/
├── {company-name}/
│   ├── account-plan.md      # Key stakeholders, strategy, timeline
│   ├── notes/               # Meeting notes, call summaries
│   ├── proposals/           # Custom proposals, pricing docs
│   └── correspondence/      # Important email threads saved
└── _archive/                # Closed deals (won or lost)
```

## When Working on Deals
1. Always check HubSpot first for the latest deal stage and activity
2. Reference the Sales Cycle Overview for stage-appropriate actions
3. For handoffs, follow the Sales→Implementation Handoff Process
4. After Closed Won, save executed contract to Google Drive: https://drive.google.com/drive/folders/1pGFyvU-YR5OUwfsEAGF8O_cR_q-1ZRgx
5. After Closed Lost, ensure the reason is detailed in HubSpot

## HubSpot Deal Stages
Discovery → Demo → Pricing/Solutioning → Compliance/Legal → Closed Won/Lost

## Key Fields to Keep Updated
Reference the HubSpot Deal Fields database in Notion for required fields at each stage.
