# Collateral Folder — Instructions

This folder stores sales decks, proposals, one-pagers, and case studies.

## Source of Truth
The master sales deck lives in Google Slides (linked from Notion: https://www.notion.so/2c74fe173c4a80689abbe67fde7c3e38). Never edit the master — create copies here for customization.

## What Goes Here
- Customized decks for specific prospects
- One-pagers and leave-behinds
- ROI calculators and business cases
- Case study summaries tailored per industry
- Proposal documents

## Deck Customization by Competitor
- **vs NetSuite:** Emphasize implementation speed, AI-native, $5M Buyout Fund, no customization entropy
- **vs Rillet:** Emphasize revenue automation depth, multi-entity architecture, full finance OS vs GL-focused
- **vs QuickBooks:** Emphasize scaling beyond spreadsheet tax, multi-entity, automated consolidation, audit readiness
- Always structure around the Three Pillars: Revenue Automation → Core Accounting → Ember AI

## Follow-Up Package (Standard)
After a discovery call, the standard follow-up includes:
1. Slide deck (customized if possible)
2. List of integrations
3. SOC 1 & SOC 2 reports
4. API documentation link
